    How to Play Equipdle

    Guess the OSRS equipment by its stats. You have 8 guesses to find the correct item.

    How it works:

    For each guess, you'll get feedback on every attribute, color-coded to help you narrow down the possibilities.

    • Correct: The attribute is an exact match.
    • Close Match: For numerical stats, this means you're close to the right value.
    • ↑Partial High: The guessed item's stat is higher than the target's.
    • ↓Partial Low: The guessed item's stat is lower than the target's.
    • Incorrect: The attribute does not match.

    Good luck!

    About Equipdle

    Equipdle is a fan-made project created by an Old School RuneScape enthusiast.

    My goal was to create a fun, daily challenge for the OSRS community that tests deep knowledge of the game's items.

    All game data is sourced from the official OSRS Wiki. This project is not affiliated with Jagex Ltd.
